提摩太后书

提摩太后书 共 2 篇讲道

📄️ 提摩太后书 4:1-2

神是活人的神，活着的人要听神的话。主很快会再来，我们在生命中做了什么，每一个人从心中想做好事，但在具体情况里，却被迫变得软弱。罪住在肉体里，激惹人的欲望，囚禁人的魂。人不认识肉体中的罪。当保罗还是扫罗时，他在宗教里，可以行假冒伪善；我们也是一样，一直活在人的眼前。信徒首先要认识自己的罪，不只是行为上的罪，还有肉体中的罪。如果主不帮助我们，没有人能得胜肉体中的罪。当祂拜访我们时，我们要认识肉体中的罪，神的话会不断光照。人有好的愿望，但即便如此，人的生命会一直变老，以死亡结束。老年的生活很残酷，害怕、恐惧、孤单、悲伤、压制。人有好的愿望，想行善，想成为一个好人，但人的身体是取死的身体，从这个身体产生了所有的问题，内在的人在受苦。我们的主马上会再来，信徒、非信徒都感受到这是一个末世，但人放弃了，信徒也放弃了。在耶稣复活以后，祂把圣灵吹入门徒，人死去的灵再次活过来。

📄️ 提摩太后书 4:14-22

撒旦像吼叫的狮子，他日夜游行，他不想让信徒成为得胜者，当主再来，他会在地上建立他的国，他需要和他同治理的人。但是信徒爱这个世界，但这个世界会逝去，我们也经历到，世界的灵一直在变化，不是一直一样的，欲望也会过去。但信徒要信神，魂要具体得救，不只是有信徒的名字，而是具体地战胜罪和死。人在犯罪以后，有邪恶的特性。首先是对神不忠诚，对神的话，不顺从，然后看到该隐有了羡慕、嫉妒、仇恨，神希望他得救，希望他活着，他本该要明白神的旨意，然后按着去做，但是他自作主张，他嫉妒弟弟亚伯，因为亚伯他明白了神的旨意，对他来说，吃喝并不重要，重要的是拯救。但是信徒很属肉体，如同该隐，本该为自己的拯救而活，但是过一个不公义的生活，嫉妒忠诚的信徒。
