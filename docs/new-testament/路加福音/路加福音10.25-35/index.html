讲道正文​

路加福音 10:25​

→ 经文

他学了很多，这个律法师很多研读经文，但他养成了坏习惯，他变得骄傲了。谁是骄傲的，他就会试探、检验别人。他嘴上说着“夫子”，心里没有谦卑，没有准备好去听、去学。
他问“我该做什么才可以承受永生”。他觉得他做一些好事，可以得永生，他想得永生。很多时候宗教化的人，认为要做好事，罪可以得赦免，这是错的。信徒把信放在一边，而是通过律法的行为过信仰生活，这个律法师问了一个错误的问题。他如果信，就不会骄傲，会准备好去听、去学。

路加福音 10:26​

→ 经文

对于律法师，这个问题是很容易的，这关于他的职业。我们带着怎样的心去读圣经是重要的。是按照圣灵吗？是按信仰和谦卑吗？或者是带着时代的灵、带着魔鬼的灵去读圣经，为了试探神。

路加福音 10:27​

→ 经文

他用几句话就总结了圣经。

路加福音 10:28​

→ 经文

如果我们这么做，那在地上的时间会得生命。主为他指了路。
这里说“你这样行，就必得永生”，不只是律法的知识，而是为了这么去做，不只是把知识传给别人。如果我们因为福音得救了，我们要去传讲福音，如果谁这么去做了，不只接受这个话，也让话语流淌出去，他会认识神的爱、神的心，他会得生命。
我们为什么要选择死亡呢？如果按照各种错的习惯、各种灵，我们会死去。这个时代很危险，只通过一个手机，世界上各种灵会进入房间，人无法入睡，也没有平安，在白天也无法正确地学习、工作。因为有各种试探在那，有各种电影、音乐。年轻人对神的话是关闭的，但对各种音乐是打开的，在那里有各种灵，人怎么能活着呢？如果生活中，某样东西有恶灵，就不需要了。如果我们接受神的灵，生命中会有平安，我们要想想，自己生命中想要什么，不想要什么。

路加福音 10:29​

→ 经文

要显明自己有理，这是问题。人活在罪中，但要证明自己正确。另一个问题就是人用无花果树的叶子遮盖，人不想面对自己的问题，反而遮盖，人的软弱就不会被医治。如果诚实地到主面前，会解决自己的问题，但如果尝试着找自我公义，自己有理，就一直有问题。
他问“谁是我的邻舍呢”，他嘴上说有爱，但他不知道谁对自己有爱，不知道爱谁。这是大多数人的情况，强调信仰是个人的，不爱邻舍，不讲福音。很多年轻人，因为罪，他们认识到人会以死亡结束，他们有死亡的知识，就想着那就随意生活吧，想怎么做就怎么做。但我们也要爱他们，爱邻舍。如果他们属灵的眼打开了，就会有生命的知识。

路加福音 10:30​

→ 经文

耶路撒冷意味着和平的根基，耶路撒冷在摩利亚山上，在这个山上，亚伯拉罕带着信仰、顺从，献上自己的儿子以撒。神看到亚伯拉罕的信，这个信，与对神的话语的顺从有关。也在这个地方，神原谅了大卫和神的子民。大卫被神试探，他数点了神的民，以色列人属于神，但大卫的骄傲被激惹，他要数点神的民，但之后大卫意识到自己的罪，神就在耶路撒冷原谅了他的罪，也在这里建立了他的殿。在耶路撒冷，人可以与神得和解，我们可以有平安，因为我们的罪被赦免。
信徒不知为何，他的信仰软弱了，他要去耶利哥。耶利哥是阻挡神的旨意、神的民的，是第一个阻挡以色列人进入应许之地的城市。耶利哥这个世界不是为了信仰的，不是为了耶稣的。虽然在信仰中有平安喜乐，但人不知为何，反对信仰。信徒到耶利哥，但在路上就被打得半死。

路加福音 10:31-32​

→ 经文

他们没有爱，对一个将死的信徒没有和解。

路加福音 10:33​

→ 经文

耶稣是被人轻视的，但他心中充满爱。

路加福音 10:34​

→ 经文

要上前，进入这个情况。当我们遇到这种情况，要上前倒酒和油。油与圣灵流淌有关，酒与生命的力量、生命的喜乐有关。
这里讲到“扶他骑上自己的牲口，带到店里去照应他”，这个小店是真正的教会，半死的人可以在那里被医治，被休养。

路加福音 10:35​

→ 经文

二钱银子是属灵的恩赐，本来无法做大事，但如果被正确地使用，人可以得平安、可以站立。
