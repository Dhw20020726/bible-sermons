马太福音 27:45-66

以下内容整理自马太福音 27:45-66 的讲道。

经文摘录​

路加福音 23:39​

→ 讲道
39 那钉十字架的犯人，有一个讥诮他说：“你不是基督吗？可以救自己和我们吧！”

路加福音 23:40-41​

→ 讲道
40 那一个就应声责备他说：“你既是一样受刑的，还不怕　神吗？
41 我们是应该的，因我们所受的与我们所做的相称；但这个人没有做过一件不好的事。”

路加福音 23:42​

→ 讲道
42 就说：“耶稣啊，你得国降临的时候，求你记念我！”

路加福音 23:43​

→ 讲道
43 耶稣对他说：“我实在告诉你，今日你要同我在乐园里了。”

马太福音 27:45​

→ 讲道
45 从午正到申初，遍地都黑暗了。

马太福音 27:46-48​

→ 讲道
46 约在申初，耶稣大声喊着说：“以利！以利！拉马撒巴各大尼？”就是说：“我的　神，我的　神！为什么离弃我？”
47 站在那里的人，有的听见就说：“这个人呼叫以利亚呢！”
48 内中有一个人赶紧跑去，拿海绵蘸满了醋，绑在苇子上，送给他喝。

马太福音 27:49-51​

→ 讲道
49 其余的人说：“且等着，看以利亚来救他不来。”
50 耶稣又大声喊叫，气就断了。
51 忽然，殿里的幔子从上到下裂为两半；地也震动，磐石也崩裂。

马太福音 27:52​

→ 讲道
52 坟墓也开了，已睡圣徒的身体多有起来的。

马太福音 27:53-56​

→ 讲道
53 到耶稣复活以后，他们从坟墓里出来，进了圣城，向许多人显现。
54 百夫长和一同看守耶稣的人，看见地震并所经历的事，就极其害怕，说：“这真是　神的儿子了！”
55 有好些妇女在那里远远地观看；她们是从加利利跟随耶稣来服事他的。
56 内中有抹大拉的马利亚，又有雅各和约西的母亲马利亚，并有西庇太两个儿子的母亲。

马太福音 27:57-61​

→ 讲道
57 到了晚上，有一个财主，名叫约瑟，是亚利马太来的，他也是耶稣的门徒。
58 这人去见彼拉多，求耶稣的身体；彼拉多就吩咐给他。
59 约瑟取了身体，用干净细麻布裹好，
60 安放在自己的新坟墓里，就是他凿在磐石里的。他又把大石头滚到墓门口，就去了。
61 有抹大拉的马利亚和那个马利亚在那里，对着坟墓坐着。

马太福音 27:62​

→ 讲道
62 次日，就是预备日的第二天，祭司长和法利赛人聚集，来见彼拉多，说：

马太福音 27:63-66​

→ 讲道
63 说：“大人，我们记得那诱惑人的还活着的时候曾说：‘三日后我要复活。’
64 因此，请吩咐人将坟墓把守妥当，直到第三日，恐怕他的门徒来把他偷了去，就告诉百姓说：‘他从死里复活了。’这样，那后来的迷惑比先前的更利害了。”
65 彼拉多说：“你们有看守的兵；去吧，尽你们所能的把守妥当。”
66 他们就带着看守的兵同去，封了石头，将坟墓把守妥当。

讲道正文​

耶稣在十字架上十个小时，他经历了人的背叛，侮辱，他被他的子民抛弃。

路加福音 23:39​

→ 经文
神给两个犯人机会可以得救。其中一个犯人，说了与祭司长和长老一样的话。我们必须有信仰，也要保守信仰，撒旦会以各种方式夺走我们的信仰，通过不同情况，氛围，或者人的话。

路加福音 23:40-41​

→ 经文
他有明白，他之前犯过大的罪，他的生活与所想的不同，他成为强盗，但在他死前，他抓住一个机会去信耶稣。

路加福音 23:42​

→ 经文
因他的罪，他必须离开属地的国，但他对神的国有盼望，虽然他是强盗，但他知道自己是有问题的人，他知道耶稣没做不好的事。神看人的心。

路加福音 23:43​

→ 经文
这个强盗在受苦中仰望耶稣，信仰成长需要很长时间，但信仰起效不需要很久。信仰生活中每个情况，如果我们仰望神，仰望主，我们要经历信主的功效，但我们要有真正的信，要有真正的态度，主不会轻看真正的信仰，心和态度。

马太福音 27:45​

→ 经文
在后三个小时，耶稣承担了所有黑暗，人的罪和死亡。然后神离开耶稣，这个黑暗长达三个小时，在耶稣生命的最后，他的呼喊是因为我们的罪，我们是不公义的人，所以不要一直寻找自己的公义，为自己辩护，我们要有更深的信仰，不需要争执，比较，无论如何都是有罪的，不义的。我们要为了弟兄姊妹，就算自己被忽视，也不需要愤怒。如果神为我们受苦，我们也为了弟兄姊妹受苦，他们过的好，我们就高兴，过的不好，我们就难过。在教会中，没有自我，我们要更多为了彼此。

马太福音 27:46-48​

→ 经文
现在人有了同情，但人的同情对人的拯救没有帮助，很多情况毫无帮助。

马太福音 27:49-51​

→ 经文
在旧约中，在圣所和至圣所中间有幔子隔开，神的拯救在至圣所里，犯罪的人没有道路可以到至圣所，到神那里。人的信仰，不那么容易得到，一个和我们一样的人，他是无罪的，但他的身体被撕裂，然后人通向永生的路打开了，是耶稣以受苦，以死亡打开了。侍奉的弟兄姊妹要更深理解十字架，这条路是神很深预备的。

马太福音 27:52​

→ 经文
这是耶稣死的权柄，他的死战胜了死亡的权势。

马太福音 27:53-56​

→ 经文
门徒逃走了，这些妇女是更刚强的，抹大拉的马利亚是很特别的，耶稣从她身体中赶出七个鬼。耶稣的门徒里没有被鬼附的，马利亚从没放弃跟随耶稣，她一直紧紧跟随。

马太福音 27:57-61​

→ 经文
这个门徒非常富有，也非常有勇气，他向彼拉多去要耶稣的身体。没有勇气，信仰不起作用，胆怯的不能侍奉神，最后他们也会背叛。

马太福音 27:62​

→ 经文
法利赛人见了许多神迹，他们见过死人活过来，但他们不信耶稣复活，他们见过神迹，但把耶稣当成迷惑人的，也把门徒当成小偷。

马太福音 27:63-66​

→ 经文
但是耶稣已经不在里面了。
