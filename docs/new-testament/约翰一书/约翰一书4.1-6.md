## 讲道正文

### 约翰一书 3:21-24
<a id="sermon-约翰一书-3-21-24"></a>
<a href="#excerpt-约翰一书-3-21-24" className="sermon-link">→ 经文</a><br/>
*第一个人亚当抛弃了神唯一的命令，人陷入罪中，人的生命中有了许多问题，没有神的话语，就没有清楚的目标和意义。在我们的生命中，如果神与我们同在，有神的话，我们就有一切。*  
*在新约，通过耶稣的死和复活，给犯罪的人新的机会，同时也给了重要的命令：所有人一定要信耶稣。如果没有这个信，就没有赦罪。人胜不过罪，在肉体中，罪就像活着的虫子，激惹着人去犯罪，特别是年轻人，在心中，无法有平安，无法有平静。如果他们很多与电子设备打交道，就会越多犯罪，心中就更没有平安，甚至无法入睡。但信耶稣的人，所有罪都被赦免，可以有新的开始。*  
*信仰不是信一次，而是我们住在耶稣里，他也住在我们里面，那就会有道路、真理和生命。我们需要这个信仰，这是神的命令，就是信他儿子耶稣基督的名。神通过耶稣做了一切，神看到我们的恐惧、害怕。特别是对于未来的恐惧不安。所以神让耶稣来到地上，结束罪和死的问题，然后他复活，进入我们。如果我们信耶稣，他就通过圣灵在我们里面，我们也在耶稣里面，这是我们的信仰。大多数信徒有错误的理解，信一次，然后想做什么就做什么，不停犯罪，但他们不呆在耶稣里，那耶稣他无法呆在让他那么多受苦的人里面。信仰不是一次，而是和耶稣联合在一起，这是新约里的诫命，首先是信，然后彼此相爱。*  
*现在所有年轻人很自私，也没有力量在生命中站立，他们怎么可能爱别人呢？人可以在想法里有许多幻想，“我是一个友好的，有爱，还不错的人”，但在具体情况中，无法去爱别人，无法从自私里走出来。特别是年轻人，为了自己的乐趣，如果可以做自己喜欢的事，如果可以满足肉体的情欲、眼目的情欲和今生的骄傲，一切别的就都不在乎了。如果可以玩游戏，那别的都随便。然后人不管自己的父母、家庭，什么都不管。很多年轻人都是这样的。人赚钱，不一定是为了满足吃喝或者解决住房的问题，很多时候是为了犯罪，没有爱是这个时代的特点。*  
*但作为信徒，我们有这个诫命：信耶稣，并彼此相爱。我们要有现实，有实际。要爱弟兄姊妹，也要爱世界上的人，那些还不信耶稣的人，因为爱，所以我们传讲福音，但撒旦一直通过肉体引诱我们，我们想传福音，肉体会阻挡，很难站起来，但我们要带着爱去得胜，带着爱去传福音。*  
*遵守诫命意味着我们待在信仰里，待在耶稣里，耶稣也住在我们里面。* 

### 约翰一书 4:1
<a id="sermon-约翰一书-4-1"></a>
<a href="#excerpt-约翰一书-4-1" className="sermon-link">→ 经文</a><br/>
人是属灵的器皿，人忙于各种各样的灵，为了被充满。当孩子还小的时候，他们不会做什么坏事，幼儿园的孩子在一起玩耍，也不感到无聊，他们也不会做太多无意义的事。但随着孩子成长，到了十三四岁，身体也在成长，各种各样的欲望也在成长，肉体的情欲变得很强，玩的游戏也不是简单的游戏了，特别是电脑游戏，有很多恶灵，所有可能的灵都在里面，有暴力的、不洁净的、狡诈的灵，在游戏、音乐的背后，也都有特定的灵，因为人是属灵的器皿。但只有信耶稣的信徒，因为罪的赦免，这个灵是神的灵，神的灵通过耶稣经历了人的生命，因为耶稣要住在我们里面，因为永生要住在我们里面，如果活在旧人里，我们胜不过罪与死，会假冒为善，所以信徒通过主的启示，会胜过撒旦的攻击，不会陷在某个罪里，被囚禁。然后我们会在信仰里成长，我们要战胜，意味着我们要顺服心中的圣灵，我们不会像过去一样，一直被迷惑、被囚禁，可以对恶灵没有反应。
