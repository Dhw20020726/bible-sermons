## 讲道正文

### 约翰一书 3:11-19
<a id="sermon-约翰一书-3-11-19"></a>
<a href="#excerpt-约翰一书-3-11-19" className="sermon-link">→ 经文</a><br/>
人也有器官，人出问题，不是器官上出了问题，医生尝试使用药物，让药物在人身上运作，但医生不认识人的生命是如何组成的。人的想法是人的一个器官，还有人的感觉，还有人的意志，最重要的是人心。
首先，人的想法。当人没有信仰的时候，人的想法只为自己的肉体，而且这个想法复杂不稳定，它是按照物质化的世界强调的运行。但我们有了信仰之后，我们认识了神的爱，神的爱在身体流淌，我们开始是为了神，为了弟兄姊妹，我们会为弟兄姊妹的拯救祷告，在想法中，充满神的旨意，甚至是没有自我的。
还有人的感觉，在旧人里，这些都不稳定，没有一个是稳定的，但在信徒中，要被神的爱充满，不要忘记耶稣是罪人的朋友，我们的主是有怜悯的。在十字架上，他怜悯了强盗，他可以和主在乐园里。我们的主看人，和人看人不同，我们的主不指指点点，希望我们的感觉中，充满神的爱，充满神的怜悯。
在人的感觉中，有人的意志，意志中有人的方向，为了达到一个目的，按旧人，我们的意志是按不义的方式运作，比如人要变的富有，活着达到好的成绩，不管以任何方式要达到目的，甚至信徒也按旧人，自作主张，甚至在神的旨意里自作主张，不知为什么，想在人面前看起来好。如果圣灵起作用，神的旨意和人就是合一的，我们会是为了邻舍，不是为了自己。
特别是我们的心,是魂中最深的器官。心要像堡垒，不能轻易被掠夺。撒旦掠夺了犹大的心，他不想和主站在一起了，他的心在财富那里，他允许魔鬼进入自己的心，然后他出卖了自己的主。人的心是信仰生活中最重要的器官。圣经上说，呼求主名的就能得救，很简单，不是所有信徒都一样。有些一生中只有一两次全心呼求主，他们可以得救。虽然他们无法完全按圣灵生活，但如果一个人在死亡前，全心地呼求主名，他可以得救。

### 约翰一书 3:12-16
<a id="sermon-约翰一书-3-12-16"></a>
<a href="#excerpt-约翰一书-3-12-16" className="sermon-link">→ 经文</a><br/>
信徒欠了爱的债，所以我们也为弟兄舍魂。具体的，我们的心、想法、意志、感觉要为神的旨意。

### 约翰一书 3:17
<a id="sermon-约翰一书-3-17"></a>
<a href="#excerpt-约翰一书-3-17" className="sermon-link">→ 经文</a><br/>
信徒可以一次做一个大事，但第二天这个爱可能就不在了，我们必须让这个爱存在心中，越来越强。在过去，我们的爱，我们的心是纯净的，会感恩，但随时间，我们只领受，不感恩了。神的爱，每一天在我们的生命中流淌，我们生命中会变得非常富有。

### 约翰一书 3:18
<a id="sermon-约翰一书-3-18"></a>
<a href="#excerpt-约翰一书-3-18" className="sermon-link">→ 经文</a><br/>
信徒不要演戏，我们会活在神的眼前，不是人的眼前，我们要具体去做。

### 约翰一书 3:19
<a id="sermon-约翰一书-3-19"></a>
<a href="#excerpt-约翰一书-3-19" className="sermon-link">→ 经文</a><br/>
其实并不容易，过去我们被人的话、想法影响着我们的意识形态，被意识囚禁了。但主在我们内在讲话，在我们内心深处有光的话，撒旦尝试把话语拿走，但话语很深地起作用了。所以我们才会来到主日，神的话语是生命的话，这个话语就是神本身。神的话语在我们里面，我们不再属肉体了，我们是属灵的，是属真理的。  
我们的主充满恩典和真理，他也想让我们的生命充满真理，就是死和复活，这就是真理。随时间人会死，人的身体回到土里，但我们有真理，就是死和复活，死亡无法控制内在的人，我们的魂会永远活着。在教会，阴间的权柄无法得胜。在我们的生命中，恩典一直流淌，不该有苦水抱怨。

### 彼得前书 1:22-23
神的爱清净了我们，而且要发展成爱弟兄的心。在今天，人被手机捆绑，甚至人爱手机超过爱人，随时间，人在属灵上会不断生病，然后死去。但对神来说，人依旧可爱，人可以变的十分美好，人可以是神的杰作。如果信徒被改变流淌神的爱，他们就变成神的杰作。如果我们传福音，要带着这个理解。

### 创世纪 1:27
神想让按照他的形象创造的人充满这地，也治理这地。在旧约中，神让两个王治理以色列，是通过神的灵，他们可以治理。  
第一个王是扫罗，他一开始很谦卑，以色列有12个支派，扫罗来自便雅悯支派。在扫罗做王之前，便雅悯支派非常的骄傲，他们有特别的战士，于是和其它支派打仗，他们应该悔改的，但因为骄傲，他们没有悔改，他们与自己的弟兄打仗，最后快被灭族了。但神想拯救这个支派，神拣选了一个王-扫罗。扫罗一开始能够认识到自己是从这样的支派出来的，他很谦卑，但随时间，他开始任性，自作主张，神想通过扫罗治理百姓。在以色列与非利士人的争战里，外在看时不可能得胜的，但神给了扫罗胜利，他应该认识到神的帮助。神也通过忠诚的仆人撒母耳在讲神的话，但扫罗抛弃神和神的话，然后，扫罗身上就有了恶灵。但神还是一直给他机会，通过撒母耳和教会，扫罗有许多机会悔改，他应该认识神，认识神的旨意，他应该是谦卑的，但他变得任性、骄傲。

### 撒母耳记上 22:11-13
扫罗因为嫉妒要杀大祭司，没有人害他。谁因为自作主张，有恶灵在他身上，他就有很多想法，幻想。  
如果信徒在旧人里，不接受十字架的话作为真理的话，就会像扫罗一样，突然就争竞骄傲，他无法听神的话语，也不可能按照圣灵生活。

### 撒母耳记上 22:14-16
本来扫罗应该通过清楚的话语得光照的。

### 撒母耳记上 22:17-19
扫罗本该按神的话，消灭亚玛利人，但相反，他消灭了神的祭司。信徒要知道自己的旧人和肉体里没有好东西。所以耶稣基督来，是带着恩典和真理。
