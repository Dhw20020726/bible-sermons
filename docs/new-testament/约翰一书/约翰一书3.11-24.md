## 讲道正文

*人也有器官，人出问题，不是器官上出了问题，医生尝试使用药物，让药物在人身上运作，但医生不认识人的生命是如何组成的。人的想法是人的一个器官，还有人的感觉，还有人的意志，最重要的是人心。*
*首先，人的想法。当人没有信仰的时候，人的想法只为自己的肉体，而且这个想法复杂不稳定，它是按照物质化的世界强调的运行。但我们有了信仰之后，我们认识了神的爱，神的爱在身体流淌，我们开始是为了神，为了弟兄姊妹，我们会为弟兄姊妹的拯救祷告，在想法中，充满神的旨意，甚至是没有自我的。*
*还有人的感觉，在旧人里，这些都不稳定，没有一个是稳定的，但在信徒中，要被神的爱充满，不要忘记耶稣是罪人的朋友，我们的主是有怜悯的。在十字架上，他怜悯了强盗，他可以和主在乐园里。我们的主看人，和人看人不同，我们的主不指指点点，希望我们的感觉中，充满神的爱，充满神的怜悯。*
*在人的感觉中，有人的意志，意志中有人的方向，为了达到一个目的，按旧人，我们的意志是按不义的方式运作，比如人要变的富有，活着达到好的成绩，不管以任何方式要达到目的，甚至信徒也按旧人，自作主张，甚至在神的旨意里自作主张，不知为什么，想在人面前看起来好。如果圣灵起作用，神的旨意和人就是合一的，我们会是为了邻舍，不是为了自己。*
*特别是我们的心,是魂中最深的器官。心要像堡垒，不能轻易被掠夺。撒旦掠夺了犹大的心，他不想和主站在一起了，他的心在财富那里，他允许魔鬼进入自己的心，然后他出卖了自己的主。人的心是信仰生活中最重要的器官。圣经上说，呼求主名的就能得救，很简单，不是所有信徒都一样。有些一生中只有一两次全心呼求主，他们可以得救。虽然他们无法完全按圣灵生活，但如果一个人在死亡前，全心地呼求主名，他可以得救。*

### 创世纪 1:27

大卫犯了极大的罪，他陷入了黑暗中，那是他忘了一切，他忘了神，忘了神的国，忘了以色列，他陷入了黑暗。他不知道自己在做什么、想什么、在计划什么，撒旦的灵掌控了他，在这次以后，撒旦继续驱使他，他将自己的仆人乌利亚杀了。  
不管是谁，人胜不过罪，胜不过撒旦的攻击。但大卫和扫罗不同。  
神以各种方式让扫罗悔改，给他机会，但他不悔改，他的心中有问题，他的骄傲很快在成长，神让他消灭亚玛利人，但他消灭了神的祭司。  
但在大卫犯罪之后，先知拿单找到大卫，对他讲了神的话，大卫被光照了。

### 诗篇 51:5
<a id="sermon-诗篇-51-5"></a>
<a href="#excerpt-诗篇-51-5" className="sermon-link">→ 经文</a><br/>
他没有说自己杀了自己的仆人，他更深地认识到自己的罪。人不明白，在犯罪后，人成了肉体是怎么回事。大卫他认识到了罪是怎么回事。不要肤浅，好像自己有天赋或者很公义，也不要像小孩子，得到玩具就觉得自己要做王了。主为了犯罪的人准备了十字架的话语，如果话语在我们心里，就流出甜水，我们可以活着、可以站立。我们的神通过大卫做了这么多，神也为他做了很多，给他很多得胜，神的恩典很多，大卫他也爱神。但如果没有十字架，当某一刻，当魔鬼来，他就被迷惑了，陷入了黑暗，他成了杀人犯。人的罪非常糟糕，如果人属肉体，人是比自己所想象的样子更遭的。他不想犯罪，他想为了神，为了神的民，他的心中不想这么做，但罪一直激惹他，罪让他做了。

### 诗篇 51:6
<a id="sermon-诗篇-51-6"></a>
<a href="#excerpt-诗篇-51-6" className="sermon-link">→ 经文</a><br/>
当人的心中没有真理，没有十字架，没有复活，人的里面是不诚实的，所以人不喜乐。我们的主想给我们最宝贵的，我相信，神让爱他的人得益处。

### 约翰福音 4:14
<a id="sermon-约翰福音-4-14"></a>
<a href="#excerpt-约翰福音-4-14" className="sermon-link">→ 经文</a><br/>
人只能想到属物质的水，当信徒不明白十字架的话，也不接受十字架的话，不接受真理，他们没有活水，不知道生命的活水。  
十字架可以让生命的泉源从心中喷涌。但魔鬼也会来，一直堵住活水井，我们要不断刨挖，让活水流淌。这是生命的争战，我们要活着。所以我们的旧人和肉体与耶稣同钉十字架，这是好的，我们可以表达神的荣耀，神想通过我们在这个时代有见证。

### 诗篇 51:7-10

### 诗篇 51:11
<a id="sermon-诗篇-51-11"></a>
<a href="#excerpt-诗篇-51-11" className="sermon-link">→ 经文</a><br/>
大卫他与亚当不同，亚当犯罪后，亚当用无花果树的叶子遮盖，躲在后面。但大卫很积极，他有真正的心，他爱神，他也认识神的爱。如果神的儿女爱神，他们就会回来，他们知道活在神的面前有多重要。  
大卫犯罪后，他面临着与扫罗一样的危险，神的灵会离开，他是有勇气的人，无论犯什么错，我们要有这样的心，神也不会收回他的灵。

### 约翰一书 3:19
<a id="sermon-约翰一书-3-19-2"></a>
<a href="#excerpt-约翰一书-3-19-2" className="sermon-link">→ 经文</a><br/>
通过信，我们属灵重生，我们属于真理，在旧人中，我们活在谎言中，我们无法正确活着。  
我们的心在神面前可以安稳，人一直在准备、工作，但人不知道真正的空间，在人的心中，若人心中有平安喜乐，有恩典和真理，有生命和盼望，他就有生命的力量。

### 约翰一书 3:23
<a id="sermon-约翰一书-3-23"></a>
<a href="#excerpt-约翰一书-3-23" className="sermon-link">→ 经文</a><br/>
这里有两个命令：信和爱。  
当我们是非信徒的时候，手和脚不断犯罪，但每次来聚会很难。信仰以后，我们的肉体也很沉重，传福音或者帮助弟兄姊妹做一些都很沉重，很多善恶。但主不断改变我们，我们心中会有爱，对弟兄姊妹，对邻人，爱会成长。在过去，我们讲许多大话，没用的话，但现在我们可以讲福音，通过此，人可以活过来。
