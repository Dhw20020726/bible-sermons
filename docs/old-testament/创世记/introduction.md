## 讲道正文

### 创世纪 2:7
<a id="sermon-创世纪-2-7"></a>
<a href="#excerpt-创世纪-2-7" className="sermon-link">→ 经文</a><br/>
神与我们的关系是比父子更近的，我们有神的灵，所以成为活的魂，人犯罪后，神的灵离开人，人只忙于地上的尘土，所以人不认识神，造物主。但是作为信徒，我们认识神与我们同在，可以看到神在我们里面，通过主耶稣生命的奉献，我们的罪被赦免。在新约，我们通过神的灵，内在持续活着。越来越多活得生命的力量。人不明白神的旨意，也不认识圣灵，只是觉得圣灵可以做神迹，但其实圣灵是赐生命的灵，让信徒内在得到生命的力量。内在与肉体不同，肉体会结束。当我们缺乏灵，我们缺乏力量，圣灵可以让我们的魂一直活着，信仰的目的是为了魂的拯救。

### 创世纪 2:8-9
<a id="sermon-创世纪-2-8-9-1"></a>
<a href="#excerpt-创世纪-2-8-9-1" className="sermon-link">→ 经文</a><br/>
神只想为人类做最好的事，信徒要明白神的心，抓住神的爱。神的心没有改变，在这个末世，神呼召我们，要祝福我们。神为人建立了生活的空间-伊甸园，是充满果实的园子，人的心，身都可以得到满足。神为人做的是最完美的，在今天也一样。神为了人没有恶的意念，完全是为了人，但人做事，只为私利，所以人做事，充满谎言和诡诈，但神是完全为了人，只有神让人真的高兴，让人的内在和外在都得满足。

### 创世纪 2:8-9
<a id="sermon-创世纪-2-8-9-2"></a>
<a href="#excerpt-创世纪-2-8-9-2" className="sermon-link">→ 经文</a><br/>
伊甸是生命的源头，在生命的水流淌的地方，有许多宝石。教会是神的田地，神的园子。在教会，永生要成长，教会是个充满果实的园子，我们是在神的葡萄园中工作的。希望在教会中的信徒都是有能力去结果子的，因此，神可以满足，信徒自己也可以满足。我们是在和神同工，人是神的田地，被建立的教会是神的田地，教会要像神的园子，果子要成长，生命的水要流淌，还需要宝石和材料。在今天，这些材料要被建造起来，神想在地上有他住的房屋，在地上和他的孩子同住。

### 创世纪 2:10
<a id="sermon-创世纪-2-10"></a>
<a href="#excerpt-创世纪-2-10" className="sermon-link">→ 经文</a><br/>
侍奉是最大的恩典，本来一个犯罪的人，失丧的人，如果我们在地上，做神想做的事，这是极大的恩典，很多年轻人，想做工人，传讲神的话，但是不聪明，留在愚拙里。愚拙是因为和魔鬼打交道了，我们需要智慧的灵。要谨慎怎么在上面建造。

### 创世纪 2:11-12
<a id="sermon-创世纪-2-11-12"></a>
<a href="#excerpt-创世纪-2-11-12" className="sermon-link">→ 经文</a><br/>
金银宝石，在创世纪里，有三种宝石，金和神的荣耀有关，本来人是地上的尘土，但是当我们领受圣灵，就成为活石，通过信仰我们被改变成为宝石。但除外还有草木禾秸，是错误的同工。草木禾秸找起来很快，造起来也很快，无法用来建造神的家，神的家不会很快建造起来，需要神长时间的工作，人才可以成为宝石，人要和圣灵完全联合，然后他才成为真正的材料，可以有真正的建造，在这些材料中有神的品格，耶稣可以做到，为罪人奉献生命，虽然他有神的形象，但他降卑自己，让自己成为奴仆，我们要做和他一样的工。

### 创世纪 2:13-14
<a id="sermon-创世纪-2-13-14"></a>
<a href="#excerpt-创世纪-2-13-14" className="sermon-link">→ 经文</a><br/>
错误的同工，无法得赏赐，希望我们可以得赏赐，不受亏损。

### 创世纪 2:15-16
<a id="sermon-创世纪-2-15-16"></a>
<a href="#excerpt-创世纪-2-15-16" className="sermon-link">→ 经文</a><br/>
谁可以与神同工呢，谁可以在做正确的工作呢，是那些过圣洁生活的人，我们的神如果一直呆在我们里面，我们就会成为神的同工。

### 创世纪 2:17
<a id="sermon-创世纪-2-17"></a>
<a href="#excerpt-创世纪-2-17" className="sermon-link">→ 经文</a><br/>
神的殿是圣的，就是我们，我们属于神，是被赎买的，在教会没有自私也不强调自己的利益，然后神的爱可以流淌，每个人都成为神的殿，成为神的同工，然后在神的园子里建造。人不能忘了神的建造，否则会活在自私里，在旧人里，失去圣灵的引领。
