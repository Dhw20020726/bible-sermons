## 讲道正文

### 创世纪 2:15
<a id="sermon-创世纪-2-15"></a>
<a href="#excerpt-创世纪-2-15" className="sermon-link">→ 经文</a><br/>
*神将伊甸园作为亚当的生活环境，亚当要做的就是修理和看守。修理在原文里也有耕种的意思，与生命的丰盛和多结果子有关。另一方面，与看守、掌管有关。亚当要保守他的生活环境。*  
*从信徒个人角度，人的心是伊甸园，在那里有永生成长，还有耶稣的形象。如果我们不保守我们的心，撒旦一直要进入我们的心，要穿透我们的心，撒旦来，是要偷窃、毁坏、杀害。但我们的心越多属于耶稣，我们的主要给我们生命，我们的内在会有喜乐。*  
*另一方面，教会也是伊甸园，主想让真正的信徒有生命的丰盛，可以多结果子，充满生命。在教会中不止有成长的信徒，还有少年，还有婴孩。首先信徒要在自己心中掌权，然后在教会中就会有神的掌管，这是神想要的。*

### 哥林多前书 5:1
<a id="sermon-哥林多前书-5-1"></a>
<a href="#excerpt-哥林多前书-5-1" className="sermon-link">→ 经文</a><br/>
淫乱就是面酵，撒旦想通过信徒的罪穿透聚会，聚会前我们要祷告，求主洁净我们的心，我们不想把面酵带到教会。

### 创世纪 2:16-17
<a id="sermon-创世纪-2-16-17"></a>
<a href="#excerpt-创世纪-2-16-17" className="sermon-link">→ 经文</a><br/>
神给了唯一的一个禁止，分别善恶树的果子你不可吃。在信仰生活中一定要注意，因为吃的日子必定死。如果违反了禁令，会失去生命力。吃了分别善恶树的果子，就会一直指责别人，看不见自己的问题，在生活中无法掌管，教会中也会有问题，嫉妒、争吵、分裂、争竞。因为违反了这个诫命，所以教会中，信徒一直有问题。人可以讲不同的理由，但唯一的原因，就是违反了这个诫命，吃了分别善恶树的果子，就失去了平安喜乐，在日常生活中也不满意，在每天的工作、学业中，我们会一直看到分别善恶树，但我们不能吃，吃了，马上死亡就进来了。

### 创世纪 2:18
<a id="sermon-创世纪-2-18"></a>
<a href="#excerpt-创世纪-2-18" className="sermon-link">→ 经文</a><br/>
在之前，每一次神表达他的心，他都说好。在创造人后，他说甚好，但是他说那人独居不好。因为神虽然是造物主，但神也有愿望，和人一起生活。神认识人和人的需求，也照顾人的需求。配偶，意味着合适的、相匹配的帮助者。很多次，年轻男人简单的，没有神的旨意，就自己做决定，按感觉、欲望，或者各种标准，自作主张的去寻找，很多时候这个妻子就不是帮助者。当我们按神的旨意生活，做决定，每件事都会被引导到好的地方。我们要坚信，只有神知道怎样配合夫妻是合适的，我们自己是不知道的。

### 创世纪 2:19
<a id="sermon-创世纪-2-19"></a>
<a href="#excerpt-创世纪-2-19" className="sermon-link">→ 经文</a><br/>
动物和人是不同的，动物只是用土造成的，但人有从神来的生气，就是神的灵。动物尽管是活的魂，但是它没有神的灵，所以我们可以看到，动物的生命和人不同，它们的注意力只在地上，飞鸟也是一样，因为他们只是从土来的，但人的关注点不止在地上，人会思考生命的意义，会思考到神还有永生。如果人只是看这个地，而没有时间看天，就像动物一样生活，那是错误的生活。  
神本不需要把这些生物带到亚当面前，但神想和人同工，神想让人掌管，在犯罪以前，亚当和神有一个想法，一个心，所以他可以认识到每个动物的特性，然后给名字，他比每个科学家更有智慧，他从神得了智慧，亚当不是按他的身体在行动，而是按神的灵。如果一个人按圣灵生活，他就可以知道神如何创造的了。神是赐生命的灵，如果人接受圣灵，也按圣灵去做，他就会有稳固的生命，他不会下沉，他会站立。在犯罪以后，人需要神的灵，真理的灵，一定要帮助我们。如果圣灵不帮助我们，我们瞬间会被撒旦欺骗。

### 创世纪 2:20
<a id="sermon-创世纪-2-20"></a>
<a href="#excerpt-创世纪-2-20" className="sermon-link">→ 经文</a><br/>
当我们随时间在信仰中成长，对神的旨意更清楚了，主会给合适的帮助者。神会带到我们面前，而不是自己来回来去地看谁是合适的，所以要忠诚地侍奉，随时间，神会带来。如果在信仰中，神为我们配合的妻子，一定是有帮助的，两个人一定属灵，如果丈夫属灵，妻子属肉体，心还在世界上，那么将不是帮助，而是负担。如果年轻人只按肉体看外在，感觉自己的心感动了，然后很快跟着跑了，很危险，那不是合适的，将一生成为负担。

### 创世纪 2:21
<a id="sermon-创世纪-2-21"></a>
<a href="#excerpt-创世纪-2-21" className="sermon-link">→ 经文</a><br/>
按神的旨意，他沉睡了，不是按自己的意思。神不是简单的从土里再造一个人，那对亚当不是合适的，而是从亚当的内在，拆一根骨头。信徒总想什么都不做，就想得神的爱和祝福。但想得到，就要有大的损失。在圣经中，沉睡和死有关，亚当在经历死后，再醒来才能得到他的妻子，不是简单的给。

### 创世纪 2:22-23
<a id="sermon-创世纪-2-22-23"></a>
<a href="#excerpt-创世纪-2-22-23" className="sermon-link">→ 经文</a><br/>
当神把动物带到亚当面前，他不是很热情，但当他看到夏娃，他很有热情。在他犯罪以前，他认识到妻子的价值，他甚至爱夏娃超过自己。  
耶稣作为末后的亚当，为我们的罪，经历死和复活，他成为神的灵，为我们预备了洁白的衣，还有掌权，他都为我们预备了。他不想自己独自作为神的继承人，他想与我们一同作为神的继承人，但耶稣的妻子一定要与他相配。耶稣是怎样的，信徒也应该一样。新郎如何，新娘也一样。为了得到相配的妻子，耶稣的肋旁被打开，我们要明白这个奥秘，更多与主相似。当教会被预备好了，主会承认，说，这终于是我的新娘。

### 创世纪 2:24-25
<a id="sermon-创世纪-2-24-25"></a>
<a href="#excerpt-创世纪-2-24-25" className="sermon-link">→ 经文</a><br/>
在犯罪以前，他们两个之间，还有他们和神之间，没必要遮盖，没有羞耻和恐惧。但在人犯罪以后，因为罪，人有许多秘密、羞耻。在人面前，有恐惧，然后想很多，想自己的脸面，假冒为善，这是因为罪。
