讲道正文​

引言​

我们的主一直与我们同在。如果他同在，就有一切。他会做一切，会给一切。信徒要看到、要认识到。但是时常看不见，也忘了神在生命里做了什么，就像以色列人，神给了那么多，但他们马上就忘了。但是有一些事情我们不能忘。我们的信仰是从亚伯拉罕开始的，通过这个信，我们称义、得救。这个信不是一次的，而是持续的，要更强。这个信是顺服的信、坚固的信，是刚强的信，甚至是可以与神同活的信仰。
人因为自己的不信和不忠诚，就把过去的一些信徒神化，称赞他们。但是亚伯拉罕，他只是一个普通人，神拜访了他，对他讲话，他按照话语行了。今天，如果神对我们讲话，我们也可以按着去做。信仰生活的原则是一样的，昨天、今天、明天，神是不变的，但是信徒不明白，不明白耶稣所做的，没有鲜活的信。

创世记 12:1​

→ 经文

神没有对他解释，没有解释。今天人为了成功，为了富有，可以离开本地本族。但是谁可以在神说的时候，可以去做呢。在亚伯拉罕的老家，人是拜偶像的，人是属灵的器皿，那么人如果没有神，人就自动拜偶像，忙于各种灵、罪和不义。
神想拯救亚伯拉罕，让他成为自己的儿女，但神看到，在这个地方是不可能的，人很容易按习惯，按文化传统生活，神想要救他。

创世记 12:2-4​

→ 经文

他带着罗德，他有自己的意愿。在他被呼召以后，一开始的十年，至少是十年，他不一定都是按照信仰去生活的，不一定都是按照信仰去做的，他自作主张地去了埃及，他的生命有了危险，失去了自己的妻子。没有人一开始就是按照信仰，但神一直忠诚地与他同在，不管他忠诚或者不忠诚，神都一直与他同在，不管他怎么做决定，神都与他同在。随着时间，有了很多问题，因为财富，他无法与罗德一同生活，他想和罗德一同生活，后面隐藏着他的不信。他要和罗德分开。
分离一直是悲伤的，神想要有合一，但是按照神的方式，以人的方式是不可能的，不管自己多少努力，无法成就。他要和罗德分开，然后罗德，他被更多问题困扰了，亚伯拉罕为了救罗德，要和那些王打仗，他的生命更危险了，但神对他非常忠诚。

创世记 15:1-2​

→ 经文

他不是太高兴，他自己在自己的生命里收集了很多问题，神一直在给他解决问题，但他还是不高兴，他的心中一直有问题。这是犯罪的人，得多少都不喜乐，因为心中有问题。

创世记 15:3-5​

→ 经文

人只是关注属地的，但是神要带领人到天上。

创世记 15:6、希伯来书 11:1​

→ 经文

很多人称自己是信徒，但真的明白信仰的人很少。信不是空的，而是有实质的，神也寻找有信心的人。如果人只寻找可见之事，那是阻挡信仰的，信仰是没法用一个看得见的东西去比较的，神是灵、是话语。

希伯来书 11:2-3、希伯来书 11:6​

→ 经文

按照我们的信，一切才会成就，如果人有信仰，会有一切，他知道他会拥有。当信仰在我们里面起作用，一切会向好的方向。

创世记 16:1-2​

→ 经文

在第15章里，他有了信，但是他无法持续地信，特别是在他的妻子给了一个建议之后。

创世记 16:15-16、创世记 17:1​

→ 经文

可能他之前13年，忙于以实玛利，他有属地的快乐，他没活在神的面前，他忘记神，随便地去做。撒旦可以带来极大的损失，已经被呼召24年了，他是如何生活的，神对他一直是忠诚的，但很难，人可以有真正的信仰。

创世记 17:2-14​

→ 经文

立约的印记是割礼。在新约，我们的割礼是在心中。如果心中没有割礼，没有十字架，信仰就不会运作，仍会按肉体和旧人而活，不会按照圣灵。如果属灵重生，心中要有割礼，不是按照过去，而是按照新人。耶稣是道路、真理、生命，虽然很窄。为什么是窄路？是因为信徒想要自作主张。

创世记 20:1-2​

→ 经文

一直到20章，神还是要很多工作。有一些品性无法很快被改变，人的魂要得救，要一生的时间。他有害怕、恐惧，他爱自己超过自己的妻子。

创世记 20:3​

→ 经文

如果神不帮助我们，我们的生命无法运作，家庭生活也无法运转。然后21章里，以撒出生了。

创世记 21:1-8​

创世记 22:1-2、创世记 22:16-18​

→ 经文

没有行为的信仰是死的。

希伯来书 5:7-9​

→ 经文

耶稣要作为人学习顺从，基督徒得了圣灵，也要学习顺从圣灵。没有顺从，就没有魂的拯救。人的肉体是着急的，是自作主张的，如果属灵重生了，但还是自作主张，没有割礼，只是浪费时间。谁按圣灵生活，他就是主的门徒，是真正的神的儿女。
