讲道正文​

如果信徒和神分开，拜偶像，尽管神有恩典、有爱，但在旧约里也看到有时候，神对他的百姓发怒。在新约，通过耶稣的血，赎买了信徒，但是神也不喜悦，如果信徒还是爱世界和世界上的事。
信徒得了圣灵，开始按照圣灵生活，也去讲福音，很多被改变，可以不再爱物质，不再爱世界。可是随时间，慢慢地，胜不过自己的肉体、欲望。开始在世界里，爱世界上的事，开始了宗教，真正的信仰不在了，生活中就会有恐惧。很多人是信徒，但是不按照圣灵，爱世界和其上的事。
但是神一直尝试，管教自己所爱的，通过管教，他们可以成为真正的神的儿女，在信仰中成长，可以结出熟杏。神的爱是强的,神不放弃人。神想掌管自己的子民，但是经常，信徒就像以色列人，想要像外邦人一样，没有神，要让人做王。

撒母耳记上 8:4-8​

→ 经文

他们出埃及以后，还是一直有埃及的思路，一直有倾向，要像非信徒一样。我们要做决定，圣灵在我们里面，但是耶稣他是否作为王在我们里面居住，他说先求神的国和神的义，然后其他所需要的会加添。信徒想要从神得祝福，但是又不想让神做王，想要自己做王，自作主张。神按照以色列人的愿望，立了一个王-扫罗，神清楚地对他讲，要杀灭亚玛利人，但是扫罗他害怕，他怕人，他是王，但是他抛弃神的话，选择按照人的话去做，按照人的意思，错误的信徒抛弃神的话，听人的话。因为没有意愿，让耶稣做王，而是要自作主张。

撒母耳记下 24:1-9​

→ 经文

在他们身边有先知，大卫，他可以求问先知，可以求问神。在历代志里写她被撒旦激惹，那么他要到神那里求问，但是他自作主张，没有神，他要数点人数。因为他里面有一些错误的爱，对世界、对世界上的事有骄傲。本来他的生命中有很大的困难，在犯罪以后，他生命中有死亡，按着肉体去做了，他要经历受苦，他要逃亡，但神让他得胜，胜过仇敌，他有忠诚的仆人，但是他旧人的品格还在，就是高傲。他老了，本来越老，他越要为神，但是他开始想寻找认可，想成名，想要有一些价值。

希伯来书 10:7​

→ 经文

耶稣是神的儿子，但是他降卑，对抗罪到流血，他明白神的心、神的旨意，他至死的顺从。
信徒不要高傲，可能神在生命中给了很多，但是我们的心和爱要在神那里，要爱神，而不是被世界、被世界上的事所充满。当大卫数点百姓，他的心中，没有了对神的爱，好像这些民是他的一样，好像他自己就做王了。就算是为了争战，他也没有必要去数点人数。仍知道神在争战，但是高傲在他心里升起了。年轻的时候因为肉体的欲望，他又大的跌倒。年老了，这个高傲就成长、升起了。

撒母耳记下 24:10​

→ 经文

在德语里是他的心击打他，他很长时间在黑暗里，有很多想法，他的心要击打他。神创造了我们内在的器官，但越来越多人的心不起作用了，只是很多善恶、想法。越是按照肉体，按照骄傲，良心就越不运作。大卫他九个月二十天在黑暗里，当他的心被今生的骄傲充满以后，他的心就长时间无法正确的运作，虽然是受膏者，但是却被魔鬼所引领。但当神光照他，他马上悔改。

历代志下 26:1-21​

→ 经文

本来他有神的敬畏，神很多帮助他。但在他强盛以后，高傲升起了。高傲的人无法听神的话。

撒母耳记下 24:11-13​

→ 经文

这是为了他的拯救，他的罪是大嘴，但是神像父亲一样，管教自己的儿子，给他三个选择，神没有随意地击打他。大卫他明白，这是一个管教，为了他的拯救。

撒母耳记下 24:14​

→ 经文

他认识神，认识神的手，认识神的恩典和怜悯。人可以演戏，好像看起来好，有怜悯，但是人和神不同，我们是否还是一直寻找人的认可，会给我们带来什么。

撒母耳记下 24:15-16​

→ 经文

神是怜悯的神，他也会后悔。

出埃及记 32:7-14​

→ 经文

神赦免，但是信徒不要按照罪，一直硬着颈项，不要高傲、发怒，而是要悔改。

撒母耳记下 24:17​

→ 经文

他有彻底的悔改，之前有许多骄傲在黑暗里，忘了神，没有了爱，但是现在，他看到的不是一个士兵，而是羊群。

加拉太书 2:19-20​

→ 经文

问题就来自于他的旧人，他的肉体。
